tqec.circuit.schedule.merge_scheduled_circuits

merge_scheduled_circuits(circuits: list[ScheduledCircuit]) → ScheduledCircuit[source]

Merge several ScheduledCircuit instances into one instance.

This function takes several scheduled circuits as input and merge them, respecting their schedules, into a unique ScheduledCircuit instance that will then be returned to the caller.

KeyError: if any of the provided circuit contains a qubit target that is

not defined by a QUBIT_COORDS instruction.

Returns:

a circuit representing the merged scheduled circuits given as input.
