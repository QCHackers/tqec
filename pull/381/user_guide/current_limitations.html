Current known limitations#

This page lists all the known limitations of the tqec library.

Note

We aim at making this page as exhaustive as possible, but it will never be 100% exhaustive. If you think that this page is missing an important entry, please raise an issue to let us know.

Important

If any of the feature discussed below is important to you, please let us know! We will try to prioritize depending on community feedback. If you do not want to wait, please have a look at the How to contribute and open a pull request.

Spatial junctions#

A spatial junction is any cube that has at least 2 pipes in the spatial (XY) plane. These kind of computation require special handling that is not currently implemented.
