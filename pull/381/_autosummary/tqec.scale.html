tqec.scale

Defines several scalable classes such as LinearFunction or Scalable2D.

This module defines the necessary classes to help with scalable structures. LinearFunction simply represents a linear function a * k + b where a and b are floating-point quantities.

This LinearFunction class is for example used to represent the shape of a Template for any input value k. More specifically, QubitTemplate has a shape that should exactly match a pair of LinearFunction(2, 2) which is basically 2k + 2.

Scalable2D is exactly made to represent such pairs of scalable quantities.
